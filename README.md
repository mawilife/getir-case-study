**Note :** You need to install **Node.Js** to run projects
- ### Installation
  ```bash
    git clone https://github.com/mawilife/getir-case-study.git
    cd getir-case-study
    npm install```
 
- ### Test
  - #### Unit Test
  ```bash
    npm run test:unit
    ```
  - #### E2E Test
  ```bash
    npm run test:e2e
   ``` 
  - #### Unit & E2E Test
  ```bash
    npm run test
    ```
- ### Run
  ```bash
    npm start
